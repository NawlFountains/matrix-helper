# Matrix helper

A simple front-end in React with Typescript for helping and visualizing steps in Matrix operations

## Setup 

1. **Clone the repository:**
   ```bash
   git clone git@github.com:NawlFountains/matrix-helper.git
   ```

2. **Enter folder project:**
   ```bash
   cd matrix-helper/
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start development server:**
   ```bash
   npm run dev
   ```

## Stack

- **React** - Frontend Library 
- **Vite** - Development environment
- **TypeScript** - Static Typing
- **Tailwind CSS** - For making my life a little easier with CSS
